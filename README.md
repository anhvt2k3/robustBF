# How to test?
## 1. For Window users (with MingW available):
- Open terminal in the OOP folder
- Run this line in your terminal
  
  `if ($?) {g++ run.cpp -o run}; if ($?) {./run}`
## 2. For Linux users (with MingW available):
- Open terminal in the OOP folder
- Run this line in your terminal
  
  `g++ run.cpp -o run && ./run`
