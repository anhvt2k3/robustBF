# How to test?
## 1. For Window users:
- Open terminal in the OOP folder
- Run this line in your terminal
  `if ($?) {g++ run.cpp -o run}; if ($?) {./run}`
## 2. For Linux users:
- Open terminal in the OOP folder
- Run these lines in your terminal
  `g++ -o myprogram run.cpp && ./myprogram`
